# REST API - STUDY📖

This was my first contact with java springboot. Through a static view this web page locally manage users of any kind, and store them in a database.

## Technologies 🔨🔧
  - Java 17
  - Springboot
  - PostgreSQL
  - Html and JavaScript

## How to run it ▶️
It's crucial to have installed in your machine: Java 17+ and PostgreSQL
  - Copy the git url and clone in your computer through terminal typing "git clone <URL_COPIED>"
  - Open the project on your IDE and run the main method inside `src/main/java/br/com/springboot/curso_jdev_treinamento/Application.java`
  - Then open in your browser the url to run the application or just run the endpoints on Postman or similar.
